# Oathbound
Oathbound is a game created in Unity for the Global Game Jam 2023, with the theme "Roots". It was made in 48 hours by some members of Vector Game Studio, a GUC game development club.

## Where to Play
[Itch Page](https://adoptedfatty.itch.io/oathbound)

[GGJ Page](https://v3.globalgamejam.org/2023/games/placeholder-2)

Download the executable build folder from either link, unzip it, and run `VGS_GGJ.exe`. Alternatively, play it directly in the browser on the itch page.
